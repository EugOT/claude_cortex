# Cortex — Bibliography

The 72-reference bibliography behind Cortex's 23 neuroscience-grounded
mechanisms and its information-retrieval pipeline. Every mechanism in the
codebase traces to one or more of these published sources — see the
per-module docstrings under `mcp_server/core/`. The two research papers in
`docs/arxiv-thermodynamic/` and `docs/arxiv-context-assembly/` draw their
focused reference lists from this consolidated set.

This file is the canonical home of the "72-reference bibliography" cited in
the README and the MCP bundle manifest.
